# 更新日志

## v1.1.0 (2026-05-08)

### 功能
- ✨ 无限循环在特定的时间内向群聊发送表情
- ✨ 支持配置最小和最大发送间隔
- ✨ 默认为启动状态，需要设置群聊

### 管理指令
- 🎮 `/random_image_start` - 启动任务
- 🎮 `/random_image_stop` - 停止任务
- 🎮 `/random_image_status` - 查看状态
- 🎮 `/random_image_send` - 立即发送
- 🎮 `/random_image_config` - 查看配置
- 🎮 `/random_image_reset` - 重置统计
