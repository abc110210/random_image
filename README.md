# AstrBot 随机时间发送表情插件

随机时间，随机发送表情包

## ✨ 功能

- **随机时间间隔** - 可以设置最小和最大间隔之间随机发送
- **自定义图片** - 从指定的文件夹里面随机选取图片发送
- **统计信息** - 统计发送数量和错误次数
- **管理命令** - 有停止，和启动，状态管理命令

## 📦 安装方法

***1.用官方的方案安装就行了***
***2.下载github压缩包再AstrBot里面手动安装***

## ️⚙️ 配置说明

在 AstrBot WebUI 中配置以下参数：

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `image_folder` | string | `images` | 图片文件夹路径（相对插件目录） |
| `min_interval` | integer | `60` | 最小发送间隔（秒） |
| `max_interval` | integer | `300` | 最大发送间隔（秒） |
| `target_group` | string | `""` | 目标群号（必填） |
| `verbose_logging` | boolean | `false` | 是否启用详细日志 |

### 管理员指令

| 指令 | 别名 | 权限 | 说明 |
|------|------|------|------|
| `/random_image_start` | `/开始发图` `/启动发图` | 管理员 | 启动随机图片发送任务 |
| `/random_image_stop` | `/停止发图` `/关闭发图` | 管理员 | 停止随机图片发送任务 |
| `/random_image_send` | `/立即发图` `/发一张` | 管理员 | 立即发送一张随机图片（测试用） |
| `/random_image_config` | `/发图配置` `/查看配置` | 管理员 | 查看当前配置信息 |
| `/random_image_reset` | `/重置统计` `/清空统计` | 管理员 | 重置统计数据 |
| `/random_image_status` | `/发图状态` `/查看状态` | 所有人 | 查看当前任务状态和统计 |

### 支持的图片格式

- JPG / JPEG
- PNG
- GIF
- BMP
- WEBP
